# Исследование возможностей чат-ботов с ИИ в добывающей промышленности

## Автор: [Байдин Илья]

## Цель исследования
Сравнить качество решения прикладных задач горнорудной отрасли с помощью зарубежного (Gemini) и отечественного (GigaChat / YandexGPT) ИИ-ассистентов. Оценить их применимость для инженеров.

## Используемые модели
1. **Зарубежный бот:** Gemini 
2. **Отечественный бот:** YandexGPT

## Выбранные кейсы
1. **Кейс 1: Диагностика неисправностей насосного оборудования**
2. **Кейс 2: Производственная безопасность (Задымление)**

## Методология
1. Сформулированы единые промпты для каждого кейса.
2. Запросы отправлены обоим ботам.
3. Результаты сохранены в папку `results/`.
4. Проведен сравнительный анализ ответов.

## Основные выводы
* **Зарубежный бот:** Продемонстрировал глубокое понимание физики процессов (связал кавитацию и износ подшипников с симптомами). Выдал четкий текст оповещения с учетом специфики производства.
* **Отечественный бот:** Показал хорошие результаты, ответил развернуто и понятно. Текст для системы оповещения получился строгим и хорошо структурированным.
* **Общий итог:** Обе нейросети отлично подходят для решения задач в добывающей промышленности. Зарубежную модель лучше использовать для сложного технического анализа (поиска причин поломок оборудования), а отечественную — для составления инструкций, чек-листов и текстов по технике безопасности.
